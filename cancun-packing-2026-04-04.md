# 坎昆出行打包清单 — **2026年4月4日** 出发

**出行人员：** 大人 ×2｜孩子 ×2（**2 岁**、**5 岁**）

**怎么分：** **随身书包** = 全程背在身上、可带进客舱（含登机路上要用的）。**箱子** = 托运拉杆箱，或大件登机箱里「不急着在飞机上掏」的东西。锂电池、充电宝、贵重证件勿托运。

---

## 随身书包

### 证件与财物（勿托运）

- 护照（每人）
- 酒店 / 度假村确认（手机离线或打印一份）
- 机票 / 登机牌（电子或纸质）
- 旅行保险信息（如有）
- 钱包：信用卡 + 备用卡；少量现金
- 驾照或 ID（若落地租车；安全座椅预订凭证可拍照存手机）

### 飞行与路上（孩子）

- **2 岁：** 纸尿裤若干片 + **湿巾**（登机换、延误用）
- **2 岁：** 零食、**吸管杯 / 学饮杯**（空杯过安检后接水）
- **5 岁：** 零食、水壶（同样可先空杯）
- 安抚物：小玩偶、安抚巾、睡前绘本（**各带一样在书包**）
- **5 岁：** 小画本、贴纸、蜡笔（不占网也能玩）
- **5 岁：** 备用**内裤 + 薄长裤/短裤**一套（弄脏/延误）
- **2 岁：** 备用**上衣 + 裤子**一套（同上）
- **2 岁：** 围兜、儿童餐具（若飞行或候机要用）

### 电子（锂电池须随身）

- 手机 + **充电线**
- **移动电源**（只能随身，不可托运）
- 耳机（含**孩子款**）
- 平板 + 已下载动画片 / 游戏
- 笔（填入境卡等）

### 健康（建议随身）

- **处方药、常备药**（退烧、过敏等；勿托运）
- 简易急救包可拆一小包：**创可贴**、少量止痛药（按需）

### 舒适与其他

- **薄外套或毯子**（全家至少一件在手边，机舱冷）
- 空**水壶**（大人孩子；过安检后灌水）
- 太阳镜、**小支润唇膏**（带 SPF；液体须符合航司/安检容量）

---

## 箱子（托运 / 大件行李）

### 衣物

- 大人：轻薄上衣 / T 恤、短裤 / 轻薄长裤（按天数 + 备用）
- 孩子：每天 **1～2 套**换洗（多备易脏）
- 睡衣 / 内衣 / 袜子（全家）
- 舒适步行鞋（大人）；**孩子防滑凉鞋 / 运动鞋**
- 额外一件薄外套（若书包已放一件，其余放箱）

### 海滩与游泳

- 大人泳衣（每人 **2 套以上**）
- **2 岁：** 游泳纸尿裤 + 泳衣（多一套换洗）
- **5 岁：** 泳衣、泳镜（可选）
- 罩衫 / 防晒泳衣（rash guard），大人 + 孩子尺码
- 人字拖 / 溯溪鞋（**孩子合脚款**）
- **沙滩包**、快干毛巾（若酒店不提供）
- 浮潜面镜（大人可选）
- 沙滩玩具（挖沙桶、铲子；也可到当地买）
- 水上安全：**救生衣 / 臂圈 / 浮力背心**
- **塑料袋 / 密封袋**（湿衣、脏衣）

### 防晒与护肤（大瓶、喷雾放箱更省事）

- 防晒霜（正装；**儿童专用**一瓶）
- 若随身只带小支，**大瓶防晒**放箱
- 晒后修复或芦荟胶
- **遮阳帽**（全家；易压可塞箱侧）
- 太阳镜（多副可放箱；贵重的可改放书包）

### 洗漱

- 牙刷、牙膏（**儿童牙膏**）
- 止汗剂 / 体香剂
- 洗发水、护发素、**儿童洗发**小瓶（若需要）
- 剃须刀
- **驱蚊水**（大瓶托运；幼儿用法遵医嘱）

### 电子与其他（非紧急）

- **转换插头**（墨西哥多为美标 A/B）
- 手机**防水袋**（海滩用）
- 书 / 较重玩具（飞行不急需的可放箱）

### 儿童用品（大件或抵达后用）

- **2 岁：** 纸尿裤 / 游泳纸尿裤（**整包余量放箱**；书包只带当天路上用量）
- **2 岁：** 护臀膏（如需）
- 轻便**推车**或**背带**（推车多**托运或登机口托运**，以航司为准）
- 向酒店确认**婴儿床 / 护栏**（无需装箱，此项为出发前勾选）

### 健康（可托运部分）

- 急救包余量、止泻药、晕船药（若乘船；儿童用药问医生）
- 体温计、退热贴（可选）

---

## 出发前（与装箱无关）

- 家中检查
- 下载离线地图 / 度假村 App
- 清空易坏食物 / 倒垃圾

---

*若只有「随身书包 + 登机箱」、无托运：把箱子里物品压进登机箱，并把所有液体控制在航司/安检限额内，剃须刀等按目的地规定处理。*
